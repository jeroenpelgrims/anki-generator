How to import this into Anki

Download the zip file and extract it somewhere.

First import the audio files.
This must be done manually.

  1. In the Anki desktop application click Tools > Check Media
  2. In the bottom left of the window, click "View Files"
  3. Copy all .mp3 audio files from the downloaded zip into this folder

Then import the notes.

  1. In the Anki desktop application, go to File > Import.
  2. Select the TSV file that is in the zip file.
  3. Select your note type. (This note type should have a field for the audio file)
  4. Match up the fields correctly in the bottom.
  5. Click import.
